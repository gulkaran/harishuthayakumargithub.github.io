Data Types

We can store data in many different types. For example, storing numeric data would require a different type than storing textual data. The basic data types we will explore today are str, int, float, bool, and list.

str - represents textual data. For example "hello world" and "102" are both of type string regardless of the content within them.

int - represents whole numbers, both positive and negative. For example -1, 0, 1 and 9824 are all integers.

float - represents decimal numbers, both positive and negative. For example 3.14 would be of type float.

bool - represent either True or False values

list - represent a list of objects. For example ["hello", 1, False, -3.14] is of type list.

type() Function

This function allows us to get the data type of any object.

type(5)  # <class 'int'>
type("600")  # <class 'str'>
type(False)  # <class 'bool'>
type(1.23)  # <class 'float'>

Converting Between Types

It is possible to convert between types. We can use type conversions or type casting to do this either implicitly or explicitly.

Implicit Type Conversion - this occurs when the Python interpreter automatically converts one type to another without developer intervention.

x = 10
y = 10.2
total = x + y

print(total)
print("x is of type: ", type(x))
print("y is of type: ", type(y))
print("total is of type: ", type(total))

Output:

20.2
x is of type: <class 'int'>
y is of type: <class 'float'>
total is of type: <class 'float'>

Explicit Type Conversion - Say we store a number within a string but want the numeric value of that as well. We can use explicit type casting to handle this ourselves. There are built in Python functions such as str(), int(), and float() to name a few. For example the int() will take in an argument such as the string "1" and convert it to 1 of type int 

pi = "3.14159"
print(int(pi) + 2)
>> 5.14159
