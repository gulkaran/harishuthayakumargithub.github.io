Python Functions

Syntax | Return Statements | Examples

Getting Started: In this lesson you will learn all about Python functions. We will explore the syntax, use of parameters and return statements, as well as some common errors students get and how to debug those.

Download Lesson Plan Here

Syntax

A function is a block of organized, reusable code that is only run once called. They can be used for specific tasks or reducing repetitive code. They let developers avoid having repeating blocks of code in a program and overall improve the organization of a program by allowing functions to be separated on more than one file.

To create a function, use the def keyword followed by a name for the function and a pair of parentheses. Any code within the function is to be indented. See the following example: 

def hello():
    print("Hello World")

Calling a Function

A function does not run by itself, but rather needs to be called inorder for it to run. To call a function, use the name followed by the parentheses and include any parameters if needed.

hello()

# OUTPUT
>> Hello World

Parameters

Functions can take in data, known as parameters. Think of parameters as temporary variables that can be used throughout the function. Your function is a set of instructions that these parameters follow that can later be mimicked by variables in your main code when you call the function.

def greeting(name):
    print("Hello", name + "!")

greeting("John")  # John is the data being passed in as the parameter of the greeting function. 

# OUTPUT
>> Hello John!

Return Statements

The return statement allows the function to return data as a result. No code after the return statement is run.

def arr_sum(x, total):
    for i in x:
        total += i
    return total
    
nums = [2, 11, 7, 4, 16]
sum = 0

sum = arr_sum(nums, sum)  # sum updates to the value the arr_sum function returned
print(sum)

# OUTPUT
>> 40

Common Errors

UnboundLocalError: local variable referenced before assignment - this error is caused when variables outside the scope of the function are being used within the function hence the refrenced before assignment remark. Either declare them as parameters or global variables to solve.

total = 1

def my_func():
    total += 1
    print(total)
    
my_func()

>> UnboundLocalError: local variable 'total' referenced before assignment 

# FIX
total = 1

def my_func(x):
    x += 1
    print(x)
    
my_func(total)

>> 2
