Type Conversions

Concept | Examples | Java Implementation

Getting Started: This lesson will cover how to convert both numerical data types and character data types.

Recall: What are Data Types?

In languages such as Java, any variable that is created is bound to a certain data type. For example, int x = 5; states that the variable x is an integer. This means that the number cannot exceed 4 bytes in size, and it also must be an integer (a whole number).

The primitive data types (the fundamental data types) in Java include: int, double, boolean, and others. String is a reference data type.

Converting Numerical Data Types

Firstly, numerical data types, such as int and double, can be converted between with little work. This can be done using casting. Casting is done by including the name of the data type you want to convert to in front of the number you want to convert.

For example, to convert double y = 15.55; to an int, the following must be done: int x = (int)y. Variable x will now have a value of 15. It is important to note that, like integer diviion, when converting to an integer, the number will always be rounded down if there is a decimal.

The places where you use casting can also create different results. The examples below work with the same numbers, but yield different results based onw here the casting is done.

1. double x = (int)5.5+3.6; will store 8.6, since only 5.5 is being cast to an int. 5 + 3.6 = 8.6

2. double y = (int)5.5 + (int)3.6; will store 8, since 5.5 and 3.6 are casted to integers first, and then they are added together. 5 + 3 = 8

3. double z = (int)(5.5+3.6); will store 9, since 5.5 and 3.6 are first added, and then the sum of the two (9.1) is casted to an int. 5.5 + 3.6 = 9.1 => (int)(9.1) = 9

Converting Character Data Types

These conversions will be using methods unique to Java, however, the same concept can be applied to other languages. For other languages, please refer to documentation on your respective language's methods for conversion.

To convert a String to a char, a method called charAt() is used. This method is a part of every String object. To call it, write the name of your String variable, followed by a dot, and then the charAt method. The parameter for the method is the index of the string that you want to get a char of (since the char data type is used for a single variable). For example, the string "Bob", has 'B' at index 0, 'o' at index 1, and 'b' at index 2.

Example:

String name = "Anderson";
char firstLetter = name.charAt(0); // 'A'
char lastLetter = name.charAt(7); // 'n'

To convert a char to a String, the String.valueOf() method is used. The parameter requires a char to be passed into it.

Example:

char letter = 'a';
String sequence = String.valueOf(letter) + "bcd"; // "abcd"

Each numerical data type has a built-in function to convert from a String to that data type. For example, to convert a String to an Integer, there is the function Integer.parseInt() in which the parameter is a string to convert from. However, the same is applicable for a Double, with the Double.parseDouble() function. All numerical data types have this function built in.

Example:

String userID = "5521239";
int convertedUserID = Integer.parseInt(userID);

Finally, to convert from numerical data types to a string, there is the String.valueOf() method where the number is passed as the parameter. Alternatively, if for example you have: Integer num = 5;, you could also use num.toString() to do the same thing. There is a negligible difference between the two. This can not be done on primitive data types (int, double, etc.), but can be used with object data types (String, Integer, Double). Fun fact: when System.out.println(); is used, the toString method is called on all the objects that are included as parameters to be printed.

int num = 543;
String num1 = num.toString(); // "543"
String num2 = String.valueOf(num); // "543"
