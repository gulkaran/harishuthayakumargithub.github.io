Integer Division and Modulus

Concept | Examples | Java Implementation

Getting Started: This short lesson will cover the fundamentals of integer division and the use of the modulus operator. We will start with the theory of it and then prove it with some examples in Java.

What is Integer Division?

An integer takes up 4 bytes of memory and is a whole number that is not a fraction. 4.5 cannot be an integer because it is not a whole number. When the quotient from division is not an integer, the concept of integer division comes in.

Concept

At the base level, when you divide and get a non-integral quotient, everything after the decimal point is thrown away. For example: 12/5 which is 2.4, which will be expressed as 2. This also means that numbers will always be rounded down and never up.

The Modulus Operator

Since integer division gets rid of any numbers after the decimal, the modulus operator (symbol: %) can be used to get the remainder of a division operation

This is done using the % symbol in place of the division symbol (/). For example the remainder of 5 divided by 3, which is 2, can be calculated through the operation: 5 % 3.

Java Implementation

This Java Implementation shows the syntax of integer division and modulus in action, and the resultant values of the operations. 

public class A1_1_1 {

    public static void main(String[] args) {
        int a = 14;
        int b = 4;
        int c = a / b;
        System.out.println(c); // 3 (the quotient, using integer division, of 14/4 is 3).
        int d = a % b;
        System.out.println(d); // 2 (the remainder of 14 divided by 4 is 2)

    }

}
